This folder contains examples of different loops in PHP:

- for-loop.php     : Demonstrates the 'for' loop.
- while-loop.php   : Demonstrates the 'while' loop.
- do-while-loop.php: Demonstrates the 'do-while' loop.
- foreach-loop.php : Demonstrates the 'foreach' loop with arrays.

Open each file in a PHP-enabled environment to see the output.
