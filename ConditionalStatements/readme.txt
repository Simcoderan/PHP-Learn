Conditional Statement Folder - PHP Examples

This folder contains example files demonstrating different conditional statements in PHP.

Files included:

1. if-statement.php
   - Demonstrates the basic 'if' condition.
   - Executes code block if the condition is true.

2. if-else-statement.php
   - Demonstrates 'if-else' condition.
   - Executes one block if condition is true, another if false.

3. if-elseif-else-statement.php
   - Demonstrates multiple conditions using 'elseif'.
   - Allows checking several conditions sequentially.

4. switch-case.php
   - Demonstrates the 'switch' statement.
   - Selects one of many blocks of code to be executed.

5. nested-if-statement.php
   - Demonstrates nested 'if' conditions (an if statement inside another if).
   - Allows checking multiple related conditions in a hierarchical manner.

How to use:
- Run these PHP files in a PHP-enabled environment (e.g., XAMPP, WAMP, LAMP).
- Open the files in a browser or CLI to see how conditional statements work.
- Modify conditions inside the files to experiment with different scenarios.
