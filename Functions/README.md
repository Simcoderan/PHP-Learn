## üìÑ File Overview

| Filename                           | Description                                                                 |
|------------------------------------|-----------------------------------------------------------------------------|

| `strict-mode.php`                  | Enforces strict type checking using `declare(strict_types=1);`.            |
| `user-defined-function.php`        | General example of creating and using a custom function.                   |
