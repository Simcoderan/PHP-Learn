PHP Functions Folder - Examples and Use Cases

This folder contains practical examples to understand how functions work in PHP. The examples range from basic to advanced:

1. basic-function.php              - Simple function with no parameters
2. function-with-parameters.php   - Function with arguments
3. function-with-return.php       - Returning values from functions
4. default-parameters.php         - Default arguments in functions
5. variable-length-arguments.php  - Handling multiple parameters
6. anonymous-function.php         - Anonymous/closure function
7. arrow-function.php             - Arrow function (PHP 7.4+)
8. recursive-function.php         - Recursive function example
9. built-in-functions.php         - Using PHP built-in functions
10. user-defined-functions.php    - Custom grouped functions
11. include-function.php          - Including functions from another file
12. require-function.php          - Require function file

Run these scripts using a local server (XAMPP/LAMP) or CLI.
